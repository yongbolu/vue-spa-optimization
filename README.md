## Build Setup

## 开发
```bash
    # 安装依赖
    npm install
    //or yarn
    # 本地开发 开启服务
    npm run dev
```

## 发布
```bash
    # 构建测试环境 带 webpack ananalyzer
    npm run build --report

    # 构建生成环境
    npm run build
```
